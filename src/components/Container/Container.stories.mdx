# Container Component

## The Container Component is used to wrap and layout all of the children components within the page.

The Container component defines global styles that will be applied to the page for a consistent UI.

The Container component wraps your components. It creates a new `div` DOM element.

```jsx
import { Container } from '@flatironschool/ui'

const theme = {
  // TBD
}

const App = () => (
  <Container theme={theme}>
    <p>Children</p>
  </Container>
)
```
